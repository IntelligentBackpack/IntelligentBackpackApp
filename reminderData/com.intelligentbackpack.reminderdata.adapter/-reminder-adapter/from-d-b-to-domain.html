fromDBToDomain

fun Reminder.fromDBToDomain(lessons: List<Lesson>, subjects: List<Subject>): ReminderForLesson

Converts a DBReminder to a ReminderForLesson. if the DBReminder is a DBReminder with the same DBReminder.fromDate and DBReminder.toDate then it will be converted to a ReminderForLessonDate. if the DBReminder is a DBReminder with different DBReminder.fromDate and DBReminder.toDate then it will be converted to a ReminderForLessonIntervalPeriod.

Parameters

lessons

The list of lessons to get the lesson from the lesson id.
